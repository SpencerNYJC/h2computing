## Python Notes for H2 Computing

#### Operators
| Operator |    Function    |              Usage |
| -------- | :------------: | -----------------: |
| +        |    addition    |    ```1 + 1 = 2``` |
| -        |    subtract    |   ``` 2 - 1 = 1``` |
| /        |    division    |   ``` 2 / 2 = 1``` |
| *        |    multiply    |   ``` 2 * 2 = 4``` |
| **       |    exponent    |  ``` 3 ** 2 = 9``` |
| //       | floor-quotient | ``` 5 // 2 = 2 ``` |
| %        |     modulo     |    ```5 % 2 = 1``` |
